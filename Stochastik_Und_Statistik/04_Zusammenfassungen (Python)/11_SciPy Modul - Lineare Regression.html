Das stats-Modul von SciPy bietet verschiedene Möglichkeiten. Unter anderen können damit diverse Informationen im Zusammenhang mit der (Linearen) Regression ermittelt werden:

from scipy import stats as ss
gradient, intercept, r_value, p_value, std_err = ss.linregress(x,y)
mit:
slope: Steigung der Regressionsgerade
intercept: Achsabschnitt der Regressionsgerade 
r_value: Korrelationskoefficient
pvalue: Zwei-seitiger p-Wert des Hypothesentest mit Nullhypothese eine Steigung von 0 
stderr: Standardfehler des geschätzten Gradienten
