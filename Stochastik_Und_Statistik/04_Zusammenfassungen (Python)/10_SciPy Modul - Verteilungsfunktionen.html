Verteilungsfunktionen können mit dem stats-Modul von SciPy-Modul genutzt werden.

Durchgängig können die PDF und CDF über die folgenden Abkürzungen aufgerufen werden:
pmf: Probability Mass Function (bei den diskreten Verteilungsfunktionen)
pdf: Probability Density Function (bei den stetigen Verteilungsfunktionen)
cdf: Cummulative Density Function 

Nutzt man mehrere Verteilungen, kann man das Modul stats gesamthaft importieren und dann die einzelnen Verteilungsfunktionen abrufen

from scipy import stats as ss
hg=ss.hypergeom(M, n, N)

oder die Verteilungen können einzelnen importiert werden, wie im Folgenden jeweils dargestellt.

Hypergeometrische Verteilung

from scipy.stats import hypergeom
hypergeom.pmf(x, M, n, N)
Out: PMF computed at x
bzw.
hypergeom.cdf(x, M, n, N)
 Out: CDF computed at x 
mit: 
M: Grundgesamtheit (bei uns N)
n: Merkmalsträger (bei uns M)
N: Stichprobengrösse (bei uns n)
 x: Anzahl Merkmalsträger in der Stichprobe

Binomialverteilung

from scipy.stats import binom
binom.pmf(x, n, p)
Out: PMF computed at x
bzw.
binom.cdf(x, n, p)
 Out: CDF computed at x 
mit:
n: Stichprobengrösse
p: Erfolgswahrscheinlichkeit

Poissonverteilung

from scipy.stats import poisson
poisson.pmf(x, mu)
Out: PMF computed at x
bzw.
poisson.cdf(x, mu)
Out: CDF computed at x 
mit: 
mu: Rate

Exponentialverteilung

from scipy.stats import expon
expon.pdf(x, 1/lambda)
Out: PMF computed at x
bzw.
expon.cdf(x, 1/lambda)
Out: CDF computed at x 
mit: 
lambda: Rate

Normalverteilung

from scipy.stats import norm
norm.pdf(x, mu, sigma)
Out: PMF computed at x
bzw.
norm.cdf(x, mu, sigma)
Out: CDF computed at x 
mit: 
mu: Mittelwert
sigma: Standardabweichung

Weitere Methoden des Moduls

stats: statistischen Kenngrössen der Verteilung: Mean(‘m’), variance(‘v’), skew(‘s’), and/or kurtosis(‘k’), z.B. 
(m,v,s,k)=binom.stats(n,p,moments='mvsk')

ppf: q-Quantil der Verteilung (Inverse der CDF), z.B.
binom.ppf(q,n,p)

median: Median der Verteilung
binom.median(n,p)

mean: Mittelwert der Verteilung
binom.mean(n,p)

var: Varianz der Verteilung
binom.var(n,p)

std: Standardabweichung der Verteilung
binom.std(n,p)

interval: liefert die Intervallgrenzen, des Intervalls das alpha% der Werte enthält
binom.interval(alpha,n,p)
