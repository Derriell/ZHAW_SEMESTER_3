matplotlib ist eine Bibliothek zum Plotten. Sie stellt zusammen mit weiteren Modulen (wie numpy und scipy) eine gute Alternative für MATLAB dar. Mittels matplotlib kann man Diagramme und Darstellungen in verschiedenen Formaten erzeugen.

Wir werden im Folgenden das Untermodul pyplot verwenden. pyplot stellt eine prozedurale Schnittstelle zur objektorientierten Plot-Bibliothek von Matplotlib zur Verfügung. Die Kommandos von pyplot sind so gewählt, dass sie sowohl in den Namen als auch in ihren Argumenten MATLAB ähnlich sind.

Es ist allgemein üblich, matplotlib.pyplot in plt umzubenennen. In ersten Beispiel nutzen wir die plot, indem wir der Funktion ein einfaches numpy-Array übergeben. plot benutzt die Werte dieser Liste als y-Werte. Die Indizes dieser Liste werden automatisch als x-Werte genommen.

import matplotlib.pyplot as plt 
plt.plot([-1, -4.5, 16, 23, 15, 59]) 
plt.show()


Dabei wir ein zusammenhängender Graph erzeugt, obwohl nur diskrete y-Werte zur Verfügung gestellt wurden. Als Werte für die x-Achse, wurden jeweils die Indizes genommen.

Indem man einen Formatstring beim Funktionsaufruf vorgibt, kann ein Graph mit diskreten Werten erzeugt werden, in diesem Bespielt mit blauen Vollkreisen ("ob"):

import matplotlib.pyplot as plt 
plt.plot([-1, -4.5, 16, 23, 15, 59],"ob") 
plt.show()


Gibt man nicht nur ein Array an, sondern ein weiteres Array für die x-Werte, dann werden im Plot jeweils die xy-Wertepaare dargestellt:

import matplotlib.pyplot as plt
# die X-Werte:
days = list(range(0, 22, 3))
# die Y-Werte:
 celsius_values = [25.6, 24.1, 26.7, 28.3, 27.5, 30.5, 32.8, 33.1]
plt.plot(days, celsius_values)
plt.show()


Die folgenden Zeichen werden in einem Formatstring in den meisten Plots akzeptiert, um den Linienstil oder die Marker zu steuern:
=============================================
'-' (Bindestrich) durchgezogene Linie
'--' (zwei Bindestriche) gestrichelte Linie
'-.' Strichpunkt-Linie
':' punktierte Linie
'.' Punkt-Marker
',' Pixel-Marker
'o' Kreis-Marker
'v' Dreiecks-Marker, Spitze nach unten
'^' Dreiecks-Marker, Spitze nach oben
'<' Dreiecks-Marker, Spitze nach links
'>' Dreiecks-Marker, Spitze nach rechs
'1' tri-runter-Marker
'2' tri-hoch-Marker
'3' tri-links Marker
'4' tri-rechts Marker
's' quadratischer Marker
'p' fünfeckiger Marker
'*' Stern-Marker
'h' Sechseck-Marker1
'H' Sechseck-Marker2
'+' Plus-Marker
'x' x-Marker
'D' rautenförmiger Marker
'd' dünner rautenförmiger Marker
'|' Marker in Form einer vertikalen Linie
'_' Marker in Form einer horizontalen Liniw

Die folgenden Farbabkürzungen sind meistens möglich:
==================
'b' blau
'g' grün
'r' rot
'c' cyan

==================

Die folgenden Abkürzungen verändern den Linienstil:
==================
'-' durchgezogenen Linie

'-.' Strich-Punkt-Linie

==================

Wir können auch mehrere Graphen in einem Plot darstellen.

import matplotlib.pyplot as plt
days = list(range(1,9))
celsius_min = [19.6, 24.1, 26.7, 28.3, 27.5, 30.5, 32.8, 33.1]
 celsius_max = [24.8, 28.9, 31.3, 33.0, 34.9, 35.6, 38.4, 39.2]
plt.plot(days, celsius_min, "y",  days, celsius_min, "oy",
            days, celsius_max, "r",  days, celsius_max, "or")
plt.show()

Dabei können wir die Achsen sowie deren Beschriftung anpassen:

import matplotlib.pyplot as plt
days = list(range(1,9))
celsius_min = [19.6, 24.1, 26.7, 28.3, 27.5, 30.5, 32.8, 33.1]
celsius_max = [24.8, 28.9, 31.3, 33.0, 34.9, 35.6, 38.4, 39.2]
plt.plot(days, celsius_min, "y",   days, celsius_min, "oy",
            days, celsius_max, "r",   days, celsius_max, "or")
 plt.xlabel('Day')
plt.ylabel('Degrees Celsius')
xmin, xmax, ymin, ymax = 0, 10, 14, 45
 print(xmin, xmax, ymin, ymax)
 plt.axis([xmin, xmax, ymin, ymax])
plt.show()


Für die Darstellung mehrerer Plots können diese mittels Subplots zusammengefasst werden in einem Plot. Dafür gibt es verschiedene Funktionen (z.B. subplot oder gridspec). Die Funktion subplot hat folgende Parameter: 

subplot(nrows, ncols, plot_number)

Wenn der Wert der drei Parameter kleiner als 10 ist, kann die Funktion mit einem Integer Wert aufgerufen werden. Das bedeutet: Statt subplot(2, 3, 4) kann subplot(234) geschrieben werden.

import matplotlib.pyplot as plt
plt.figure
plt.subplot(231) # äquivalent zu: plt.subplot(2, 2, 1)
 plt.text(0.5, # x-Koordinate, 0 ganz links, 1 ganz rechts
         0.5, # y-Koordinate, 0 ganz oben, 1 ganz unten
          'subplot(2,3,1)', # der Text der ausgegeben wird
         horizontalalignment='center', # abgekürzt 'ha' 
          verticalalignment='center', # abgekürzt 'va'
         fontsize=10, 
          alpha=.5) # float (0.0 transparent bis 1.0 undurchsichtig)
plt.subplot(233,facecolor="#476042")
 plt.text(0.5,0.5,'subplot(2,3,3)',ha='center',va='center',fontsize=10,color="y")
plt.subplot(235)
 plt.text(0.5,0.5,'subplot(2,3,5)',ha='center',va='center',fontsize=10,color="b")
plt.show()
